Shovel Mobile App

- Clear Your Way -


Introduction

Shovl is a mobile application that provide snow clearing service. This app only acts as a mediator between actual shoveler and the customer that needs their snow removed from their property. This app will be a meeting place for the customers with snow clearing needs and the snow cleaners. This app is implemented using React-Native. This was done as a group project.

My Role

This was done as a group project and I was part of the five memeber team. I designed the information architecture of this project. I was also responsible for implementing the home page of this Application and navigation of the React app.
